## Added

- **Find - Search For Items** - Search for Items in containers by opening a container and pressing `Ctrl + F`
- **Show Me Your Skin!** - Hide player armor or make it transparent so you can see their skin under their armor.
  - `J` - Toggle mod
  - `U` - Mod settings
- **LambdaBetterGrass Refabricated** - Better grass textures
- **Wavey Capes** - Give capes physics!
- **Better Third Person** - 360 degree cam for third person
- **Chest Colorizer** - `Shift + Right-click` chests with dye to color code them.
- **Effect Timer Plus** - Shows how much time is left on a potion effect on the icon itself.
- **DeathView** - Third person view of you dying.
- **Block Meter** - Measuring blocks
  - `\` - Select your current tool as a block meter
  - `]` - Block Meter options (after it’s turned on)
- **Respackopts** - Allows us to give you server resource pack options in the future.
- **PicoHUD** - Displays the coordinates, day, and biome on your screen.
  - `Right Alt` - Toggle PicoHUD on and off.
- **Variants-CIT** - Extra resource pack support

## Added Resource Packs

- **Myth's Better Hotbar** - Cleaner hotbar
- **GUI Revision** - Pleasing container UI
- **Xali’s Potions** - Makes potions more identifiable
- **Xali’s Enchanted Books** - Makes enchanted books more identifiable
- **Os' Cleaner Signs** - Cleans up sign textures
- **Os' Colorful Grasses** - Makes grasses more immersive and fun
- **Os’ Shulkers** - Adds patterns to colored shulkers
- **Better Carpets** - Makes carpets spill over the block a little

## Added Shaders

- **Insanity Shaders** - If you want a spooky look

## Removed

- **Chest Search Bar** - Replaced with the find mod
- **Immersive: Tooltips** - “Was causing issues with custom items
- **BetterGrassify** - Replaced with the LambdaBetterGrass Refabricated mod
